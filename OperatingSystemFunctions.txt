Windows is an operating system with many features.
One of which is Plug and Play which automatically detects drivers for devices.
This is nice because it saves the hassle of having to configure your own devices. 

Windows allows for both a GUI and Command-line input method.
The GUI for Windows is very intuitive for all users.
The command promt allows for full configuration and analytics of your computer.

Another function of Windows is a seperation of background and foreground apps.
This allows you to have many programs open at once.
Depending on the program, it can be frozen or active in the background.
This allows your PC to save memory for active tasks while not losing progress in other programs.
